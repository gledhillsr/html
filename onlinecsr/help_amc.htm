HELP SCREEN (Adds, Moves & Changes)

1) New Request This button adds new adds or change orders to the system.
2) Admin This takes you to the administrator page. Admin privileges are required.
3) Line Inventory This takes you back to the Line Inventory Manager.
4) Archive This database keeps a record of all your previous change orders.
5) Edit This button takes you to the edit page where you an change existing orders,
6) BTN This is the Billing Telephone Number on you account. You can sort by BTN but you can not modify this field.
7) Service Describes what type of change is being made. Add, Move, Change, Fix, Inquire or Other
8) Requested By The name of the individual who created this request.
9) Date Created The date this request was created.
10) Instructions Instructions as to what needs to be changed or ordered.
11) Sent to  Who was this request sent to.
12) Response When the person who received this request places their response here.
13) Order # If an order or tracking number is generated then it is placed here for reference.
14) Remind Date  The remind me date will remind the person who is placing the order that it is not complete. This keeps orders and changes from falling in the cracks.
15) Status This field shows the current status of each order. Open, In Progress or Completed. A corresponding dot red, yellow or green respectively will be placed on the line inventory manager on the respective BTN.
16) Last Modified This shows the date when this request was last modified.
