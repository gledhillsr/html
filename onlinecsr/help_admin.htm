HELP SCREEN (User Manager)

1) New User This link will take you to a new page where you will add a user to your account.
2) Line Inventory This button takes you back to the main Line Inventory page.
3) Adds, Move & Changes This link will take you to the page where you make add services r make changes to existing services.
4) Edit This button allows you to change or delete each user in your system.
5) Login Name This is the login name each individual is assigned by the system administrator.  This field can not be modified.
6) Password The password is set by each user when they login for the first time.
7) Customer ID Internally all your files are in this folder. This field can not be modified.
8) User Name This is the full name of the user.
9) Admin This field show who has admin privileges.  Only those with admin privileges can view this page
10) Notify About Logins When anyone assigned to your account logs in to the system those who are marked here will be notified by email.
11) Notify About Changes When there are adds moves and changes made on your account, who do you want to be notified.
12) Department If you have several people with access to your account when an email is generated the department is also mentioned in the email to help you associate where the request/order came from.
13) Email The email address of this individual.
14) Phone  The phone number of this individual.
15) Mobile The cell phone number of this individual.
16) Access Count This shows you how many times this person has logged in to the system.
