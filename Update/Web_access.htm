Accessing iFolder Using a Web Browser

To access your Novell® iFolderTM files from a browser, right-click the iFolder icon, click iFolder Web Site, and then click Login (for PDA access) or Applet. You log in using your user iFolder ID and password, and enter your iFolder encryption pass phrase.

The iFolder Web site displays the network files and folders currently accessible to you. You can use many of the same conventions for expanding and contracting folders and opening files that are available in Windows* Explorer. To create new folders or copy, delete, rename, move, upload, or download existing files or folders, click on the down arrow next to the file or folder.

Local files and folders are not accessible using this interface. Use the file directory interface on your workstation to view local files and folders.

Click Folder View to display folders in another column and allow you to expand and contract folders. The Text View only displays the files and folders in the current directory and does not allow you to expand or contract folders.
