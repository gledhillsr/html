Accessing the iFolder Web Site

To access the iFolder Web Site, right-click the iFolder icon, then click iFolder Web Site. This launches your default Internet browser and opens it to your corporate Novell® iFolderTM Web site. This corporate portal is where you will normally download the iFolder Client and local iFolder documentation, and access your iFolder files using a Web browser.

Additional Information

For more information, as well as synchronization tips and tricks, see the iFolder End User Guide located at the Novell Documentation Web site.
