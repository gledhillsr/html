Changing Your Local iFolder Directory

Novell® iFolderTM allows you to choose the location of your local iFolder directory during the installation of the iFolder Client. However, if you already have an iFolder account with files in it, you must move or copy your files to the new location in order for iFolder to synchronize properly with the new iFolder Client.

To move your files to the new location:

  1. When the installation is finished, double-click the iFolder shortcut (to the old location) on your desktop.

  2. Click Edit > Select All.

  3. Click Edit > Cut.

  4. Close the window.

  5. Double-click the iFolder shortcut (to the new location) on your desktop.

  6. Click Edit > Paste.

For more information about iFolder features, see the iFolder End User Guide located at the Novell Documentation Web site.

To move the location of you iFolder directory after you have installed or upgraded the iFolder Client:

  1. Log in to iFolder.

  2. Right-click the iFolder icon in the system tray, then click Account Information > File > Set iFolder Location.

  3. Move your iFolder files from the old location to the new location.

    This will prevent iFolder from downloading your entire account to the new location.

  4. Log in again to iFolder.
