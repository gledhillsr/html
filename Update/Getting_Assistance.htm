Getting Assistance

Your Novell® iFolderTM administrator maintains an iFolder Web site with information about your iFolder service. To access this site, right-click the iFolder icon, then click iFolder Web Site.

For more information on using iFolder, and for synchronization tips and tricks, see the following documents, available at the Novell Documentation Web site:
