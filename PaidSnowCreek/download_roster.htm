Download the Roster for a Palm Pilot®

You can download the full patrol roster in a format that can be imported into your Palm Pilot's address book.

Instructions:  (Yes, I know this is NOT really simple)

1) You MUST have the Palm Pilot desktop software

2) Click the download button to save the roster on your hard drive (remember exactly where you put it)

3) Start the Palm Pilot's Desktop program

4) In the Desktop program select the Address tab

5) If you need to, create a new address group (ie. Ski Patrol), and go to that group
    If you are updating your group, I suggest deleting all the old names BEFORE importing new names

6) import the file you downloaded, as a comma delimited text file

7) then sync with your Palm Pilot
